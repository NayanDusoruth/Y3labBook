Cluster theoretical overview (overview provided by Scarlett, rewritten here for convenience)

Scarlett wrote a quick summary of her theoretical findings which are renotated here.

She focused on characterising clusters in the Ising model, so outlined the following cluster observables:

  • Cluster size distribution
  • Cluster shape
  • Spatial correlations
  • Time autocorrelation
  • Connection between clusters and correlations.

Cluster size distribution

\(z(n) = \frac{N_{cluster}(n)}{N}\)                   Eq 2.1 Cluster size density $N_{cluster}(n) = $ Number of clusters of size \(n\) $N = $ number of spin states in ising model

\(z(n)\) represents the “density” of clusters of a given size n in an ising model. Mean cluster size follows:

\(<n>_{cluster} = \frac{\sum n^{2}z(n)}{\sum nz(n)}\)                    Eq 2.2 Average cluster size

Which apparently has behaviour analagous to magnetic susceptibility The number of clusters per lattice point:

\(<C>_{site} = \sum z(n)\)                    Eq 2.3 number of clusters per site

With the largest cluster fraction acting as an order parameter:

\(f_{max} = \frac{n_{max}}{N}\)                   Eq 2.4

Note that at critical temperature, \(z(n)\) approaches:

\(z(n) \approx \frac{1}{n^{2+\frac{1}{\delta}}}\)                   Eq 2.5 Critical cluster density $= $ critical exponent of magnetisation-field response (2D case has \(\delta = 15\))

Cluster shape:

It is useful to define a “cyclomatic number” for any cluster :

\(c=l-n+1\)                    Eq 2.6 cyclomatic number $n = $ number of spins in a cluster $l = $ number of bonds between spins in a cluster

Given this, some idea of “compactness” may be defined and illustrated:

\(A=\frac{c}{(n-1)^2}\)                    Eq 2.7 Coefficient of compactness

Spatial correlations:

Spin correlations are described by:

\(C(r)=<s(0)s(r)>=\frac{1}{N}\sum s_{i} s_{i+r}\)                   Eq 2.8 equal time correlation function

At non critical temperatures this experiences rapid decay:

\(c(r) \approx e^{\frac{-r}{\zeta}}\)                   Eq 2.9 Non critical correlation $= $ correlation length

\(\zeta\) is related to criticality, so as the model approaches criticality it grows, increasing the correlations between distant lattice points.

Time autocorrelation:

Time autocorrelation is analagous to spatial correlations as Eq 2.7, but across some time interval \(\tau\) instead of space \(r\):

\(A(\tau) = \frac{<(M_{t}-\overline{M})(M_{t+\tau}-\overline{M})>}{(M_{t}-\overline{M})^{2}}\)                   Eq 2.10

and typically decays as:

\(A(\tau) \approx e^{\frac{-tau}{\tau_{c}}}\)                   Eq 2.11

Connections between cluster and correlations:

Spin correlations and cluster connectivity are directly related as:

\(<s_{i}s_{j}>=P(i<->j)\)                   Eq 2.12 spin correlations and probability of connection between two points

paper specifics:

Used a MC 3D ising model to generate a set of “sample models” and associated “replica models” over a range of temperatures. The replica models were used to decide when a model had equilibrated by comparing the magnetisations of 2 models; one which started all spin up, one which started all spin downs. When the magnetisations entered each other’s uncertainties; equilibration was determined.

Each sample/replica measured:

  • \(<M> =\) average magnetisation
  • $ <M^2>$
  • \(<E> =\) average energy
  • \(<E^2>\)

Which yield the following observables:

\(C_{ave} = N\frac{(<E^2> - <E>^2)}{T^2}\)                   Eq 2.13 average specific heat capacity

\(m_{ave}=|<M>|\)                    Eq 2.14 Average magnetisation

\(\chi_{ave} = N\frac{(<M^2> - <M>^2)}{T}\)                   Eq 2.15 Average susceptibility ee \(\chi_{dis} = N<M>^2\)                   Eq 2.16

Which relate to their critical exponents as:

\(t = T - T_c\)                   Eq 2.17 Temperature relative to critical temperature \(T_c\)

\(T^2C_{ave} = L^{\frac{\alpha}{v}} \overline{C}(tL^{\frac{1}{v}})\)                    Eq 2.18

\(m_{ave} = L^{\frac{\beta}{v}}\overline{m}(tL^{\frac{1}{v}})\)                    Eq 2.19

\(T\chi_{ave} = L^{2-\eta}\overline{\chi}(tL^{\frac{1}{v}})\)                    Eq 2.20

\(\chi_{dis} = L^{\eta - \overline{eta}}\chi_{dis}(tL^{\frac{1}{v}})\)                    Eq 2.21

\(\alpha =\) Specific heat critical exponent

\(\beta =\) order parameter exponent

\(v =\) Correlation length exponent

\(\eta, \overline{\eta}\) describe “power law decay”

\(\gamma = (2-\eta)v =\) susceptibility exponent

\(\overline{C}, \overline{m}, \overline{\chi_{ave}}, \overline{\chi_{dis}}\) are scaling functions for their respective observables.

The paper found the critical exponents for \(C_{ave}\) by acknowledging that \(\overline{C}\) has a maxima \(x_{0} = (T_{0}-T_{c})L^{\frac{2}{v}}\) for some \(T_{0}\) where \(T^{2}C_{ave}\) is maximised; so by finding \(T_{0}(L)\) over a range of \(L\), the exponent \(v\) and critical temperture can be fitted with:

\(T_{0}(L) = x_{0}L^{\frac{-1}{v}} + T_{c}\)                    Eq 2.21

So the free parameters \(x_{0}\), \(v\) and \(T_{c}\) can be found by polynomial fit.

The paper implies similar procedure may be used for the other observable parameters.

The paper then discusses these exponents in a wider context that i’m not fully versed in, and at the moment not fully interested in. The outline in measuring critical exponents was the main point of interest for this lab at this point.
